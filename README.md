  - [ ] ...

## Techstack

### Frontend

- Language: Javascript/Typescript
- Framework: NextJS
- Style: TailwindCss, Shadcn, LucideIcon,...

### Backend

- Language: Go
- Framework: Go-gin
- Database: MongoDB
